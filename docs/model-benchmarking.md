# Model Benchmarking

## Purpose

The benchmarking workflow compares multiple regression models using the same target variable and dataset.

---

# Supported Models

## Linear Regression

A simple baseline regression model.

## Decision Tree Regressor

A tree-based model capable of learning non-linear relationships.

## Random Forest Regressor

An ensemble model using multiple decision trees.

---

# Evaluation Metrics

## MAE

Mean Absolute Error.

Measures average prediction error.

## RMSE

Root Mean Squared Error.

Penalizes larger prediction errors.

## R² Score

Measures explained variance and model fit quality.

---

# Benchmark Output

The dashboard displays:

- Actual vs predicted visualization
- Model comparison chart
- Metrics comparison table
- Best-performing model
